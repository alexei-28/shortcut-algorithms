### Уровень 1: Основы
Упражнения для закрепления материала о базовых структурах данных: массивах, строках и связных списках.

#### Задание 1
Варианты ответов:

- O(1)
- O(log (n))
- O(n)
- O(n*log (n))
- O(n2)

Какова временная сложность следующих операций:

#### Массивы:
#### 1. Вставка элемента в конец массива
```
int[] arr = {1, 2, 3, 4, 5};
// или ArrayList<Integer> arr = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));
arr.add(6);
```
#### Ответ: arr.append (6) — O(1) амортизированная. 
В большинстве случаев это просто добавление в конец. Иногда требуется расширение массива — O(n), но это происходит редко.

#### 2. Обращение к элементу массива по индексу
```
arr[0]
```
#### Ответ: arr[0] — O(1). Прямой доступ по индексу.

#### 3. Вставка элемента в начало массива
```
arr.insert(0, 0);
```
#### Ответ: arr.insert (0, 0) — O(n). Нужно сдвинуть все элементы вправо.

#### 4. Проверка на наличие элемента в массиве
```
arr.contains(5);
```
#### Ответ: 5 in arr — O(n). Линейный поиск по массиву.
